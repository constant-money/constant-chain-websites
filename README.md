# Constant websites

## Prerequisite

Install these first:

- Docker >= 18.0.0
- Yarn
- NodeJS

## Getting Started

- Update your host file (located at `/etc/hosts` on mac) to `./blancer/hosts`
- Run `yarn balancer`
- Open each sub project (`user`, `portal`, etc.) and run `yarn install`, `yarn dev` or `yarn start`

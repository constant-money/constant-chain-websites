# Constant websites

## Core

Core is dir constants styles and components which is useful and reusable for Constant websites

## Balancer

### Requirement

- Docker >= 18.0.0

### Hosts

Change your hosts file for development

```
balancer/hosts
```

Suggestion: `gas mask` (https://github.com/2ndalpha/gasmask)

### Run

```bash
yarn balancer
```
